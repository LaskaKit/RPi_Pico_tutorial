## Rozložení pinů

![RPi pico pin layout](images/pico_layout.png)

* 26x 3.3V GPIO pinů
* 2x SPI
* 2x I2C
* 2x UART
* 3x 12-bit ADC
* 16x PWM
* 8x PIO (programovatelné input/output piny)
